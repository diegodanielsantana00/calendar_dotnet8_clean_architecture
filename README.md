# Agenda de Contatos - .NET 8 Clean Architecture

## Funcionalidades

- CRUD completo para contatos (Criar, Ler, Atualizar, Excluir)
- Validação de dados via FluentValidation
- Autenticação e autorização via JWT com 2 Regras (Membros e Admin)
- Identity para controle de Usuarios
- Mapeamento de objetos
- Persistência com Entity Framework Core 8 e suporte a migrações
- Arquitetura modular baseada em camadas (Domain, Application, Infrastructure, API)
- Preparado para deploy com CI/CD com Deploy em Docker
- Implementação de Docker Compose

### Pré-requisitos

- [.NET 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
- Banco de dados PostgreSQL
- RabbitMQ
- Docker
